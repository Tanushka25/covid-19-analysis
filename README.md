COVID-19 Data Analysis

Project Overview

This project analyzes COVID-19 data to understand infection trends, recoveries, and deaths across different regions. The analysis helps identify patterns and visualize the impact of the pandemic using data-driven insights.

Dataset

- COVID-19 Dataset (Excel)
- File: covid19.xlsx

Tools Used

- Microsoft Excel
- Data Analysis
- Data Visualization

Key Insights

- Analysis of confirmed COVID-19 cases
- Comparison of deaths and recoveries
- Trend identification and reporting
- Data cleaning and preprocessing
